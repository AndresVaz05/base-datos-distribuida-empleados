# base-datos-distribuida-empleados
Proyecto académico para la materia Sistemas de Bases de Datos Descentralizadas. Implementación de una base de datos distribuida en modalidad de espejo (replicación maestro‑esclavo) con MySQL. Incluye una interfaz web desarrollada en PHP con actualización automática vía AJAX para gestionar empleados de una empresa de tecnología. 

# Base de datos distribuida - Sistema de empleados

Proyecto para la materia **Sistemas de Bases de Datos Descentralizadas**.  
Implementación de una base de datos en espejo (replicación maestro-esclavo) con MySQL y una interfaz web en PHP que permite:

- Agregar empleados.
- Listar empleados (actualización automática cada 5 segundos vía AJAX).
- Eliminar empleados.

## Tecnologías
- MySQL (replicación)
- PHP
- HTML, CSS, JavaScript (AJAX)
- XAMPP (Apache)

## Requisitos
- MySQL Server 8.0
- XAMPP (solo Apache)
- Red local con IPs estáticas
